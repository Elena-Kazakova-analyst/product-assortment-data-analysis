# 📦 Анализ товарного ассортимента (Product Assortment Analysis)

Анализ данных о продажах розничного магазина для классификации товаров и разделения на **основной** и **дополнительный** ассортимент (для прикассовой зоны).

---

## 📋 Описание проекта
- **Источник данных:** 6737 записей о продажах (дата, покупатель, заказ, товар, количество, цена)
- **Цель:** сегментировать товары по категориям и выделить ключевые позиции
- **Период анализа:** октябрь 2018 — октябрь 2019

---

| Инструмент | Назначение |
|------------|------------|
| Python (Pandas, NumPy) | Обработка и анализ данных |
| Matplotlib, Seaborn | Визуализация |
| Yandex DataLens | BI-дашборд |
| SQLite3 | Работа с БД |

---

## 🚀 Этапы работы

### 1. Предобработка данных
- Преобразование дат в формат `datetime`
- Обработка дубликатов (966 записей)
- Добавление признаков: выручка, день недели, месяц, год

### 2. Исследовательский анализ (EDA)
- Графики количества покупок и выручки по дням
- Гистограмма распределения заказов по сумме
- Выявление пиковых дней и сезонности

### 3. Классификация товаров
- Разработаны правила для 10 категорий на основе ключевых слов
- Категории: Растения, Хоз. товары, Посуда, Декор, Инструменты и др.

### 4. Сегментация ассортимента
- **Основной ассортимент:** Топ-10 товаров по выручке в каждой категории
- **Дополнительный ассортимент:** остальные товары (для прикассовой зоны)
